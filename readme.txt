A Wordpress plugin can update and edit Woocommerce products price via CSV file as easily.

== Description ==

**Woocommerce CSV Price** lets you to edit and update Woocommerce products price via CSV file as easily.

= Update products price with one CSV =
Easily update and edit all products price via one file in 3 seconds!

= Export CSV products =
This plugin can export all products with ID,Name,Price in a CSV file for edit and archive.

== Installation ==

For install **Woocommerce CSV Price** plugin you should do these following steps:

1. Upload the plugin files to the `/wp-content/plugins/woo-csv-price` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. If you have seen error from plugin, just go to pages and create a page with name and slug "arya-csv-export".
4. Use the plugin and enjoy!
